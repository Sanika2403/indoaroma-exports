IndoAroma Exports

A full-stack spice export management web application developed for MCA Web Engineering Lab.

Features:-

- User Signup & Login Authentication
- Admin Panel
- Product Enquiry System
- Contact Us Form
- Shopping Cart
- Quantity Management
- Razorpay Payment Gateway Integration
- PDF Invoice Generation
- MongoDB Database Integration
- Live Form Validation
- Responsive UI Design

Technologies Used---------

Frontend :-
- HTML5
- CSS3
- JavaScript

Backend:-
- Node.js
- Express.js

Database:-
- MongoDB

Other Tools:-
- Razorpay Payment Gateway
- PDFKit for Invoice Generation

Modules:-

- User Authentication
- Product Management
- Cart System
- Payment System
- Enquiry Management
- Contact Management
- Invoice Generation

Payment Gateway:-

Razorpay test mode is integrated for secure online payment simulation.

Invoice System:-

After successful payment, a downloadable PDF invoice is automatically generated.

Academic Purpose:-

This project was developed as part of the MCA Web Engineering Laboratory.
